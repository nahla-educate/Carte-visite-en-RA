# Carte-visite-en-RA
carte visite en réalité augmentée en utilisant Vuforia developpement kit

## Features
* Virtual buttons(localization/ mail/ Phone/ site) 
* Play video

## Version
unity 2020.3.26.

## Installation

* Install unity
* Download this project
* Configure vuforia development kit.
* Run it.
